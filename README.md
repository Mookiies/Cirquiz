# cirquiz

A pass-and-play local multiplayer trivia game for iOS and Android.

## Prerequisites

- [Node.js](https://nodejs.org/)
- [Yarn](https://yarnpkg.com/)
- [Expo Go](https://expo.dev/go) app on your device, or a simulator/emulator

## Setup

```sh
yarn install
```

## Running

```sh
yarn workspace mobile start     # Start Expo dev server
yarn workspace mobile ios       # Run on iOS simulator
yarn workspace mobile android   # Run on Android emulator
```
