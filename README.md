# cirquiz

A pass-and-play local multiplayer trivia game for iOS and Android.

[<img src="https://play.google.com/intl/en_us/badges/static/images/badges/en_badge_web_generic.png" height="50">](https://play.google.com/store/apps/details?id=com.cirquiz.app&hl=en_US)                    [<img src="https://developer.apple.com/assets/elements/badges/download-on-the-app-store.svg" height="50">](https://apps.apple.com/us/app/circleq/id6760278898)   

## Prerequisites

- [Node.js](https://nodejs.org/)
- [Yarn](https://yarnpkg.com/)
- [Expo Go](https://expo.dev/go) app on your device, or a simulator/emulator

## Setup

```sh
yarn install
```

## Running

```sh
yarn workspace mobile start     # Start Expo dev server
yarn workspace mobile ios       # Run on iOS simulator
yarn workspace mobile android   # Run on Android emulator
```
